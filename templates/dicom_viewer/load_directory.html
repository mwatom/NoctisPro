How to Use

  • Windows: Use paths like C:\DICOM\Studies or D:\Medical_Images
  • Linux/Mac: Use paths like /media/usb/dicom or /home/user/dicom
  • Flash Drive: Insert your USB drive and navigate to its mount point
  • CD/DVD: Insert disc and use the disc mount path
  • The system will recursively scan subdirectories for DICOM files (max 15 levels deep)
  • Supported file extensions: .dcm, .dicom, .dic, and files without extensions
  • Capacity: Can scan up to 5000 files and process up to 500 DICOM files per session
  • Timeout: Scanning will timeout after 60 seconds, processing after 2 minutes for safety
  • Large Datasets: For machines with 2000+ images, the system will process the first 500 files and show a summary
Example Paths:
Windows USB: D:\DICOM_DATA
Windows CD: E:\PATIENT_STUDIES
Linux USB: /media/username/USB_DRIVE/dicom
Mac USB: /Volumes/USB_DRIVE/medical_images
